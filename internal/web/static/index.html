Supported: JSON, YAML, TOML, CSV, TSV

JSON Format Example

[
  {
    "name": "Alice Johnson",
    "notification_type": "email",
    "contact_info": ["alice@example.com"],
    "exclusions": ["Bob Smith"]
  },
  {
    "name": "Bob Smith",
    "notification_type": "email",
    "contact_info": ["bob@example.com"],
    "exclusions": ["Alice Johnson"]
  }
]

YAML Format Example

- name: Alice Johnson
  notification_type: email
  contact_info:
    - alice@example.com
  exclusions:
    - Bob Smith
- name: Bob Smith
  notification_type: email
  contact_info:
    - bob@example.com
  exclusions:
    - Alice Johnson

CSV Format Example

name,notification_type,contact_info,exclusions
Alice Johnson,email,alice@example.com,Bob Smith
Bob Smith,email,bob@example.com,Alice Johnson
Carol Davis,slack,@carol,
David Wilson,email,david@example.com; david.alt@example.com,Carol Davis

💡 Use semicolons to separate multiple emails or exclusions within a cell

TSV Format Example

name	notification_type	contact_info	exclusions
Alice Johnson	email	alice@example.com	Bob Smith
Bob Smith	email	bob@example.com	Alice Johnson
Carol Davis	slack	@carol
David Wilson	email	david@example.com; david.alt@example.com	Carol Davis

💡 Tab-separated format, great for Excel. Use semicolons for multiple values.

TOML Format Example

[[participants]]
name = "Alice Johnson"
notification_type = "email"
contact_info = ["alice@example.com"]
exclusions = ["Bob Smith"]

[[participants]]
name = "Bob Smith"
notification_type = "email"
contact_info = ["bob@example.com"]
exclusions = ["Alice Johnson"]
